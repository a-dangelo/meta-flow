Workflow: expense_approval
Description: Submit expense reports with automated fraud detection and approval routing based on amount thresholds
Access: employee
Category: hr

Inputs:
- amount (float): Expense amount in USD
- date (string): Date of expense in YYYY-MM-DD format
- category (string): Expense category (travel, meals, office, equipment, other)
- department (string): Employee department
- receipt_url (string): URL to uploaded receipt image
- employee_id (string): Employee identifier

Steps:
1. Validate receipt exists and is readable
2. Extract receipt metadata using OCR
3. Check if amount matches receipt total
4. Run fraud detection analysis on expense pattern
5. If amount is greater than 10000, route to senior manager for approval
6. If amount is between 1000 and 10000, route to direct manager for approval
7. If amount is less than 1000, auto-approve
8. Send notification email to employee with approval status
9. Update expense tracking system with new entry

Outputs:
- approval_status (string): Status of approval (pending, approved, rejected)
- approval_reference (string): Unique reference number for tracking
- estimated_reimbursement_date (string): Expected date for reimbursement
